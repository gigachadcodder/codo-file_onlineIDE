# Codo File - Code Edior with extending features.
  
  ## Installation 
    1. Open Command Prompt\ Terminal

    2. git clone https://github.com/faizan619/Codo-File.git

    3. cd .\Codo-File\

    4. npm install

    5. cd .\frontend\

    6. npm install

    7. npm i gsap

    8. cd..

    9. cd .\backend\

    10. npm install

    11. cd..

    12. npm run both
